    Multiton

    1. Histórico de versão

    Versão Data Descrição Autor
    0.1 20/03/2022 Criação do documento Daniel Barcelos, Thiago Mesquita e Vinicius Saturnino

    2. Definição

      O padrão Multiton é um padrão de projeto criacional e é uma extensão do Singleton. Ele permite instanciar um número limitado de objetos de uma classe.

    3. Aplicação

      O Multiton foi utilizado no FrontEnd com o objetivo de utilizar os componentes do React, tornando possível instanciar diversos componentes iguais com informações distintas.
